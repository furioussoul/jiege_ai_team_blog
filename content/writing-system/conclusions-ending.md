# 收尾模式（Conclusions）

> 收尾的作用：让读者记住你说的，激发行动
> 原则：最后一句话最重要

---

## 常用收尾模式

### 模式1：呼应式

**公式**：开头提到X → 结尾再提X，但意思深化了

**例子**：
```
开头：
"This is what post-AGI feels like."
这是 Karpathy 在桑拿房里发的推文。

结尾：
"This is what post-AGI feels like."
看到这句话的时候，我在想：也许我也该跑一个自己的 Loop 了。
```

**适用场景**：
- 心得体会
- 深度思考
- 有始有终

**模板**：
```
开头：[引用/观点]
...（中间展开）...
结尾：[同样的话]，但[你的理解/行动]
```

---

### 模式2：行动式

**公式**：总结观点 + 给出行动建议

**例子**：
```
这周最大的感悟是：编程这个词已经不准确了。

我不知道这对我来说意味着什么。
但我知道一件事：

我不应该再试图"写得更快"。
我应该学会"指挥得更好"。
```

**适用场景**：
- 实用类文章
- 方法论分享
- 呼吁行动

**模板**：
```
所以，[总结]。
我不应该[A]。
我应该[B]。
```

---

### 模式3：问题式

**公式**：留下一个问题，引发思考

**例子**：
```
Karpathy 说编程在过去 2 个月的变化，比过去几十年加起来还大。

我在想：
如果所有人都能用 AI 写代码，那什么才是核心竞争力？
```

**适用场景**：
- 深度思考
- 开放式结尾
- 引发讨论

**模板**：
```
[总结]。
我在想：[问题]？
```

---

### 模式4：金句式

**公式**：用一句话总结全文，制造传播点

**例子**：
```
不是比谁敲得快，而是比谁"指挥"得更好。

---

"This is what post-AGI feels like."
—— 这可能是 2026 年最让人不安的一句话。
```

**适用场景**：
- 观点输出
- 金句传播
- 强化记忆

**模板**：
```
[金句]。

--- 
[引用]
—— [你的评价]。
```

---

### 模式5：未来式

**公式**：总结现状 + 展望未来

**例子**：
```
时代在变，我要变的是适应它，而不是抵抗它。

我不想当那个还在敲代码的"工匠"。
我要学会当那个指挥 AI 干活的"指挥官"。

也许，这是程序员最好的时代。
```

**适用场景**：
- 趋势分析
- 个人成长
- 鼓励读者

**模板**：
```
时代在变，[你的态度]。

不想[A]。
要学会[B]。

也许，[展望]。
```

---

## 收尾的高级用法

### 用"对比"强化

```
开头：Karpathy 从去年12月到现在，一行代码都没写过。
结尾：我离这个状态也不远了。
```

读者会感觉到：**从"他"到"我"，故事完整了。**

### 用"选择"强化

```
我不想当那个还在敲代码的"工匠"。
我要学会当那个指挥 AI 干活的"指挥官"。
```

读者会感觉到：**你有态度，有选择。**

### 用"悬念"强化

```
也许我也该跑一个自己的 Loop 了。
```

读者会想知道：**然后呢？**
