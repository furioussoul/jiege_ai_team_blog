## 写作系统的核心组件

### 1. 素材收集

建立素材库，随时记录灵感：

- 使用 Notion 或 Obsidian 管理素材
- 定期整理和分类
- 建立标签体系

### 2. 结构模板

使用模板加速写作：

```markdown
## 文章模板

1. 标题（吸引眼球）
2. 引言（抛出问题）
3. 正文（分层论述）
4. 案例（实证支撑）
5. 总结（行动号召）
```

### 3. 写作流程

标准化的写作流程：

```
选题 → 大纲 → 初稿 → 修改 → 发布
```

## AI 辅助写作

### 适合 AI 辅助的场景

- 头脑风暴
- 大纲生成
- 润色修改
- 格式转换

### 不适合 AI 的场景

- 原创观点
- 个人经历
- 情感表达

## 工具推荐

| 用途 | 工具 |
|------|------|
| 写作编辑 | VS Code + Markdown |
| 素材管理 | Notion / Obsidian |
| AI 辅助 | Claude / ChatGPT |
| 发布平台 | 自建博客 / Medium |
