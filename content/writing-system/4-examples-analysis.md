## 拆解方法

### 三步拆解法

```
Step 1: 读一遍，感受整体
Step 2: 再读一遍，分析结构
  - 开头用了什么模式？
  - 中间怎么展开的？
  - 结尾怎么收的？
Step 3: 记录金句，思考怎么用
```

## 拆解示例

### 示例：Karpathy 的 autoresearch README

**开头模式**：反差式 + 场景式
```
"One day, frontier AI research used to be done by meat computers..."
```
用"meat computers"自嘲，制造反差，引发好奇。

**展开结构**：问题-解决-结果
```
问题：以前科研要人来做
解决：现在 AI 可以自己跑实验
结果：700个实验，2天，20个优化
```

**金句**：
```
"This is what post-AGI feels like."
"All LLM frontier labs will do this. It's the final boss battle."
```

**可学习的点**：
1. 用游戏术语（final boss）让技术人秒懂
2. 数据具体（700、2天、20）
3. 自嘲幽默（meat computers）
