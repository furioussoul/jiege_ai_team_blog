# 开头模式（Hooks）

> 开头的作用：让读者想继续看下去
> 原则：3秒内抓住注意力

---

## 使用方法

1. 看到好文章的开头，分析它用了什么模式
2. 把模式抽象出来，存到这里
3. 写文章时，从库里选一个模式

---

## 常用开头模式

### 模式1：反差式

**公式**：A说X，但B发生了Y

**例子**：
```
他2025年10月说"AGI至少还要10年"。
5个月后，他说"我们已经进入Post-AGI时代"。
```

**适用场景**：
- 观点变化
- 打脸现场
- 预测失败

**模板**：
```
[时间A]，[某人]说"[观点X]"。
[时间B]，[某事]发生了——[结果Y]。
```

---

### 模式2：场景式

**公式**：描述一个具体场景，让读者身临其境

**例子**：
```
这周刷到 Karpathy 在播客上说了一句话，让我彻夜难眠。
"I don't think I've typed like a line of code probably since December."
```

**适用场景**：
- 分享经历
- 引出故事
- 表达感受

**模板**：
```
[时间]，我看到/听到[具体内容]。
[我的反应]。
```

---

### 模式3：问题式

**公式**：抛出一个问题，引发好奇

**例子**：
```
为什么 Karpathy 说自己从去年12月到现在，一行代码都没写过？
他是不会写了吗？还是不想写了？
```

**适用场景**：
- 探讨现象
- 深度分析
- 引发思考

**模板**：
```
为什么[现象]？
是[A原因]？还是[B原因]？
```

---

### 模式4：数据式

**公式**：用数据制造冲击

**例子**：
```
700个实验，2天，110次代码修改。
这是 Karpathy 的 AI agent 在他睡觉的时候完成的。
```

**适用场景**：
- 展示成果
- 制造对比
- 强调效率

**模板**：
```
[数据1]，[数据2]，[数据3]。
这是[背景/原因]。
```

---

### 模式5：引用式

**公式**：引用名人名言，建立权威

**例子**：
```
"This is what post-AGI feels like."
这是 Andrej Karpathy 在桑拿房里发的推文。
```

**适用场景**：
- 引入话题
- 借势权威
- 制造悬念

**模板**：
```
"[原话]"。
这是[某人]在[场景]说的。
```

## 进阶：组合使用

高手的开头往往是组合拳：

```
引用式 + 反差式：
"This is what post-AGI feels like."
Karpathy 在桑拿房里发了这条推。
而他的 AI agent，刚刚自主工作了 12 小时。

数据式 + 问题式：
700 个实验。2 天。0 次人工干预。
AI 科研时代已经来了吗？
```
